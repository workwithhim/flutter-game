## üíª Configuration Steps
1. Cloning the repository:

```
$ git clone https://github.com/ahmedgulabkhan/SnakeGameFlutter.git
```

2. Open the project:

`$ cd SnakeGameFlutter`

3. Now run the app on your connected device (using terminal):

`$ flutter run`
